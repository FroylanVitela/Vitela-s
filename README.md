## Backend

El directorio `server` contiene una API Express conectada a MongoDB con esquemas de Mongoose para productos, conjuntos de opciones, guías de tallas, paquetes de prendas y políticas.

Para poblar la base de datos con datos de ejemplo:

```bash
cd server
node seed.js
```

## Frontend

El directorio `client` contiene una aplicación React creada con Vite.

```bash
cd client
npm install
npm run dev
```
